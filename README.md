# SistemaCinema 🎬

Sistema de gerenciamento de cinema em Java + Swing. Fiz pra praticar a mesma coisa do MeuPet, mas dessa vez gerenciando um cinema.

Consegue cadastrar filmes, gerenciar sessões, vender ingressos, controlar clientes e gerar relatórios de vendas. Bem prático mesmo.

## O que tem

- Cadastro de filmes
- Gerenciamento de sessões e horários
- Venda de ingressos
- Controle de clientes
- Relatórios de vendas
- Interface gráfica com Swing

## Como está organizado

Mesma estrutura do MeuPet - Controller, Model, View e DAO. Tudo separadinho pra ficar fácil de mexer.

## Como rodar

```bash
git clone https://github.com/naclarra/SistemaCinema.git
cd SistemaCinema
```

Abre no IntelliJ, roda o Main.java e pronto.

## Tecnologias

Java, Swing, MVC, DAO, IntelliJ IDEA.

## Por que fiz

Consolidar aprendizado em POO, padrões de design e interface gráfica. Basicamente repetir o que aprendi no MeuPet mas em outro contexto.
